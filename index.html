Por que as tags são importantes

A linguagem de marcação de hipertexto (HTML) é uma linguagem padrão de exibição de arquivos utilizadas na internet. E muito do conteúdo que consumimos ao navegar na rede nada mais é do que a transmissão desses arquivos, que são interpretrados pelo navegador.

Percebe-se que o dominio dessas linguagem, e de seus componentes são essenciais para um bom desenvolvimento de conteúdo que agrade o usuário.

Aqui serão abordadas algumas das principais tags que melhoram a navegação do usuário na internet

Tags h's Indice

As tags h's definem o cabeçalho de da página, de certa forma dividem o texto em sessoões, onde o h1 é a sessão mais importante, enquanto o h6 é a menos importante. Essas tag's tem característica semântica, não sendo somente usadas para modificação estética do texto. São suportadas por todos os navegadores e apresentam uma restrição: Somente deve haver uma tag h1.

Tag p Indice

A tag p é utilizada para escrever parágrafos no texto. Frases, conteúdos textuais extensos, normalmente fazem uso dessa tag. Essa tag também é suportada por todos os navegadores.

Tag mark Indice

A tag mark é uma marcação visual do texto, pode ser usada para deixar o fundo do texto com uma coloração diferente. Muito similar ao marca-texto em cadernos escolares.

Tag small Indice

A tag small é um recurso do HTML que permite reduzir o tamanho de letras, palavras, frases e até mesmo textos, desde que sejam abraçados pela tag. O comportamento dessa tag também pode ser conseguido utilizando-se css, reduzindo o tamanho da font de uma classe em específico.

Tag i Indice

A tag i é usada para deixar letras, palavras, frases e textos em itálico. Normalmente, essa tag é utilizada para denotar termos técnicos, frases de outra linguagem, pensamentos e apelidos. É suportada por todos os navegadores.

Tag u Indice

A tag u sublinha elementos textuais, podendo ser aplicado desde letras até textos completos. Deve-se evitar seu uso quando sua utilização gerar confusão com a tag link

Tag strong Indice

A tag strong marca o texto em negrito indicando que é um assunto, tópico ou até mesmo palavra importante no texto. Antes de sua existência isso era feito pela tag b. Entretanto, a tag strong tem significado semântico, enquanto a tag b somente marca em negrito.

Tag ol Indice

A tag ol gera uma lista ordenada de itens a partir de um valor inicial, que á atribuido pelo atributo starter. Pode ter sua ordenação numérica alterada pelo atribuito type, que permite que sejam feitas listas com números romas e letras na ordenação.

Exemplo

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  1. Item 21
  2. Item 22
  3. Item 23
  4. Item 24
  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Tag ul Indice

A tag ul também define uma lista, entretanto, essa listagem é feita sem ordenação, e com as marcações dos ítens feitos com pontos, quadrados e circulos vazios.

Exemplo

Com pontos

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Com quadrados

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Tag li Indice

A tag li define os elementos de uma lista ordenada ou não ordenada.

Tag a Indice

A tag a é usada para definir links entre outras páginas do mesmo site ou de sites externos, seus principais atribuitos são:

  • Href: Usado para definir o link de destino
  • Target: Usado para definir em qual aba do navegador o link de destino vai abrir

Tag hr Indice

A tag hr defini uma linha separadora de conteúdo na página de exibição do HTML

Tag sub Indice

A tag sub faz altera o alinhamento e o tamanho das letras do fragmento textual que ela abraça. Pode ser usado para subindices e para fórmulas químicas.

Exemplo: H2O, CaCO3

Tag sup Indice

Similar a tag sub, entretanto a posição do fragmento textual é sobrescrito. Pode ser usado, para expressões matemáticas

Exemplo: Texto normal Texto sobrescrito, x2+2x3

Tag font Indice

A tag font era utilizada no HTML 4 para alterar as características visuais da fonte exibida, tais como família,cor e até tamanho. Essa tag foi descontinuada no HTML 5 e essa formatação foi destinada ao CSS

Tag del Indice

A tag del é usada para mostrar correções no texto, similar ao feito em textos de redações

Exemplo: Texto errado Texto correto

Tag p Indice

A tag p é utilizada para conteúdos textuais, onde é escrito o que o usuário deve ler.

Tag abbr Indice

A tag abbr é utilizada para abreviações onde ao passar o mouse sobre a escrita sublinhada com pontos (alteração visual feita pela tag) é mostrado o valor referente a abreviação
